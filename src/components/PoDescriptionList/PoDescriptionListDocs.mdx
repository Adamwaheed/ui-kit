### With action

Action click just returns the string passed `action: "Update",`. If you want to add more functionality, and have it return an object, you can do that by:

```js
let descriptionListItems = [
	{
		title: "Dob",
		description: "1972-02-05",
		action: {
			label: "Update",
			href: "/callcentre/reports/reportname",
		},
	},
];
```

Where only the `label` is required, and you can add as many data as you want to `action` object, the click event will return the object.

### Loading

`is-loading=true` default is true

**_Important:_** When adding is loading you may want to define `itemsCount`. This is used to display the loading placeholders and the default is set to 4. The number of items to show.

### Adding content via slot

<Source
	language="js"
	dark
	format={true}
	code={dedent`

      <template v-slot:content>
        <div>

          <dd>Description</dd>
        </div>
        <div>
          <dt>Title 2</dt>
          <dd>Description 2</dd>
        </div>
        <div class="link">
          <dt>Title 3</dt>
          <dd>Description 3</dd>
        </div>
      </template>
    </PoDescriptionList>
  `}
/>

Adding .link class to description item adds a hover effect.
