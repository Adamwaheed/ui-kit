# Action bar component

import { Story, ArgTypes, Source, Meta } from "@storybook/blocks";

import * as PoActionBarStories from "./PoActionBar.stories";

import dedent from "ts-dedent";

IMPORTANT: The Action bar component was originally designed to be used at the top of a page to control all actions on the page. However, in later versions of the UI Kit, we have moved away from this design and do not recommend using the Action bar for this purpose.

### Action bar

### With highlights

You can pass an optional `highlighted: true` to `items` to highlight the button. This is useful in cases where important action buttons are added to action bar.

### With danger

You can pass an optional `danger: true` to `items`. This is useful for delete action and related actions.

### Explanation

<ArgTypes of={PoActionBarStories} />

<Source
	language="js"
	dark
	format={true}
	code={dedent`
    import { useRouter } from 'vue-router';
    const router = useRouter();
    let actionBarItems = [
        {
            label: "New Staff",
            icon: PlusIcon,
            highlighted: true
        },
        {
          label: "Delete it",
          icon: TrashIcon,
          danger: true,
        },
        {
            label: "Experiment",
            icon: BeakerIcon,
        },
    ];
    function handleActionBarClick(item) {
        switch (item) {
            case "New Staff":
                router.push('/staff/create');
                break;
            case "Experiment":
                console.log("exp");
                break;
            case "back":
                router.go(-1);
                break;
        }
    }
  `}
/>
