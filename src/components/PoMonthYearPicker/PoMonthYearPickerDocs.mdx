### Min Date and Max Date

You can set either a Minimum date that users can pick or a maximum date, or both. If you don't set a `minDate` or `maxDate` it takes +- 10 years from the current year as maximum and minimum.

<Story of={PoMonthYearPickerStories.MinMax} />

<Source
	language="js"
	dark
	format={true}
	code={dedent`
    <PoMonthYearPicker
        label="Contribution Month"
        minDate="27-02-2023"
        maxDate="01-10-2024"
        v-model="modalValue" 
    />
  `}
/>

### Display Format

You can change the display format of the date by passing `display-format=""` prop.

Refer to dayjs formating documentation. https://day.js.org/docs/en/display/format

Remember this input only returns month and year, so you can only use formating for month and year.

<Story of={PoMonthYearPickerStories.DisplayFormat} />

<Source
	language="js"
	dark
	format={true}
	code={dedent`
    <PoMonthYearPicker
        label="Contribution Month"
        display-format="MMMM YY"
        v-model="modalValue" 
    />
  `}
/>
