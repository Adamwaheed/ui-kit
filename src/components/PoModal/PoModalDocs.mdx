When openBtnLabel is empty, modal open button is not displayed. You can use a custom button to open it.

<Source
	language="js"
	dark
	format={true}
	code={dedent`
    const showNotificationBodyModal = ref(false);
    <PoModal :show="showNotificationBodyModal" @modal-closed="showNotificationBodyModal = false" modal-title="Email" id="investment-notification-status">
        <template v-slot:content>
          <div class="p-5">
            ...
          </div>
        </template>
    </PoModal>
  `}
/>

Remember to always use @modal-closed event to reset the show prop variable.
