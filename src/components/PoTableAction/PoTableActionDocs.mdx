### Custom Icon Buttons

You can use custom icons for table action. PoTableAction only accepts HeroIcons at the moment. HeroIcons 24 Outline looks best.
When using custom buttons use `btn-type="icon"` and pass the icon to `:btn-icon="XMarkIcon"`. Remember to also set a Label to the button to use as a tooltip `label="Cancel" and set an icon color using tailwind stroke color coded `icon-color="stroke-red-400"`.

<div className="po-p-5 po-flex po-items-center po-space-x-3">
	<Story of={PoTableActionStories.IconCustom} />
</div>

<Source
	language="js"
	dark
	format={true}
	code={dedent`
  <PoTableAction
    btn-type="icon"
    :btn-icon="CheckIcon"
    label="Approve"
    icon-color="stroke-sky-400"
  />
`}
/>
