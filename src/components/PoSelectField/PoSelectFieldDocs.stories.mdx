You can use `error-message` to show an error message. The field will update visually to indicate that it has an erorr. You can use `message` property to display a message to the user.
You can use the `:disabled="true"` prop to disable the field.

Every time a user selects from the dropdown `selected` event is fired.

<Source
	language="js"
	dark
	format={true}
	code={dedent`
	<PoSelectField
			label="Select field"
			:list="selectFieldListTwo"
			@selected="handleSelectFieldTwoClick"
			v-model="selectFieldTwoSelected"
		/>
		
    const selectFieldSelected = ref(null);
    const selectFieldListTwo = ref([
			{ id: 1, name: "Sub category name", subtitle: "Category" },
			{ id: 2, name: "Running around in circles", subtitle: "Activity" },
			{ id: 3, name: "Something drinkable", subtitle: "Water" },
		]);

    function handleSelectFieldTwoClick(val) {
    	console.log(val)
    }

`}
/>
