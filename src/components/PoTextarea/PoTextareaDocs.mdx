# Texarea component

import { Story, ArgTypes, Source, Meta } from "@storybook/blocks";

import * as PoTextareaStories from "./PoTextarea.stories";

import dedent from "ts-dedent";

By default the type is set to `text`. You can use `v-model` bindings in `PoTextarea`.
