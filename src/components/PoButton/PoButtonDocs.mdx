Using icons. Default button 'type button' and types link and simple can have custom labels using slots, so you can add icons, etc.
When overriding button type `simple` colors, you only need to pass `text-.. hover:text-..` You can add `:is-loading="true"` when you want to show loaidng inside the button.

<br />
<Story of={PoButtonStories.Bolt} />
<br />
<Source
	language="js"
	dark
	format={true}
	code={dedent`
    <PoButton type="simple" to="/deathreporting/dispute">
      <template v-slot:label>
        <span class="po-flex po-items-center po-space-x-1">
          <BoltIcon class="po-w-4 po-h-4 po-stroke-current" />
          <span>File a dispute.</span>
        </span>
      </template>
    </PoButton>
  `}
/>
