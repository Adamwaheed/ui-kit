# PoFilter component

import { Story, ArgTypes, Source, Meta } from "@storybook/blocks";

import * as PoFilterStories from "./PoFilter.stories";

import dedent from "ts-dedent";

This component sets filters as URL Query params on the browser. And on filter button click it emits the event `@button-click`.

You can change Filter Button Label by setting `btn-label="New Label"`.
