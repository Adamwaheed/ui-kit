# Input field component

import { Story, ArgTypes, Source, Meta } from "@storybook/blocks";

import * as PoInputFieldStories from "./PoInputField.stories";

import dedent from "ts-dedent";

By default the type is set to `text`. You can use `v-model` bindings in `PoInputField`.

### Normal

<div className="po-max-w-sm">
	<Story of={PoInputFieldStories.Normal} />
</div>

<Source
	language="js"
	dark
	format={true}
	code={dedent`
    <PoInputField
        label="Fullname"
        id="form-fullname"
        v-model="form.fullname"
    />
  `}
/>
