  Story,

<Description of={PoInputField} />

By default the type is set to `text`. You can use `v-model` bindings in `PoInputField`.

### Normal

<Canvas withToolbar>
  <div class="po-max-w-sm">

  </div>
</Canvas>

<Source
  language="js"
  dark
  format={true}
  code={dedent`
    <PoInputField
        label="Fullname"
        id="form-fullname"
        v-model="form.fullname"
    />
  `}
/>
