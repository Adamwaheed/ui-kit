# Page title component

import { Story, ArgTypes, Source, Meta } from "@storybook/blocks";

import PoPageStats from "./PoPageStats.vue";

import * as PoPageStatsStories from "./PoPageStats.stories";

import dedent from "ts-dedent";

This component is used to display stats on the page header right side of `Page Title` component.
We use this when the right side empty on pages that doesn't have action button at the top so that It won't be empty.
